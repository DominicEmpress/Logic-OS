# Logic-OS
Simulador de circuitos electronicos. Componentes: palancas, botones, luces, luces de colores, compuertas AND XOR NAND NOT... compuertas personalizadas,  compuerta BLOCK, D-flipflops Tflipsflops JKflipsflops, guardar y cargar projecto.
Gracias por descargar el projecto aqui hay una breve explicacion de como crear un circuito.
abre el .html y ya estaras en la pagina aqui tendras varios menus de componentes, flips flops y archivo si quieres guardar tu projecto o cargarlo simplemente vas a archivos y te saldra una lista con lo que hay si quieres poner algun componente vas a componentes o flips flops
y ya dentro podes elejir que componente poner, los circulos blancos son las entradas y los circulos negros son las salidas para conectarlas simplemente apreta el click en alguna entrada o salida y desapretalo en otra salida o entrada y se va a conectar, con la X podes borrar el componente y con el emoji celeste lo podes girar a la derecha, eso es todo despues sigue tu imaginacion crear circuitos lo que quieras.
